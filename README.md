# WhatShouldICite
WhatShouldICite（这段话该引谁） 是一个科研写作辅助工具，采用编辑器内Agent形态，帮助作者在写作过程中快速判断是否需要引用，并给出引用类型和检索关键词建议。 核心特点：全局可用：跨应用使用，快捷键Ctrl + Shift + C触发；不打断写作：选中文本后立即显示建议，无需切换窗口；智能分析：支持规则判断（免费）和LLM判断（更准确）两种模式；专业规则库：100+学术关键词，覆盖10种引用意图类型。设计理念：这不是文献生成器，而是写作辅助Agent。它不生成具体论文标题或作者，而是： 判断是否需要引用、建议应引用哪类工作、提供检索关键词、帮助作者在写作过程中快速获得引用建议，提升写作效率。
